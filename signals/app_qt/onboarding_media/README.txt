Медиа для интерактивного обучения (онбординга)
================================================

Сюда (или в ~/.signals/onboarding) кладите изображения и видео для шагов
обучения — миниатюра появится прямо в подсказке, по клику открывается крупно.
Если файла нет — вместо миниатюры будет написано, куда его положить; само
обучение от этого не сломается, просто пройдёт без картинки.

Поддерживаются изображения (.png/.jpg/.jpeg/.bmp/.gif) и видео
(.mp4/.avi/.mov/.webm/.mkv). Видео требует QtMultimedia (обычно есть).

Имена файлов, которые ищет обучение
-----------------------------------
Базовый тур (калибровка антенн):
  intro.png            — общая иллюстрация/заставка
  hantek.png           — прибор Hantek со встроенным генератором
  fork_k3.png          — калибровочная вилка К-3
  antenna_mount.png    — установка антенны в вилку (момент 3.5)
  adapter.png          — подключение переходной платы по ключам
  devices_found.png    — список найденных приборов
  add_subject.png      — добавление предмета в таблицу
  fork_hang.png        — вилка с антенной свободно висит в воздухе
  reference_signal.png — ОБРАЗЦОВЫЙ сигнал (эталон формы записи)
  verdict.png          — пример столбца «Годность антенны»

Продвинутый тур использует те же файлы (в основном intro.png и
devices_found.png).

Можно заменить любой .png на одноимённое видео (например reference_signal.mp4):
обучение покажет видео, если найдёт его по имени с видео-расширением.

Приоритет папок: сначала ~/.signals/onboarding, затем эта встроенная папка.
