Папки для DLL прибора Hantek (HTHardDll и сопутствующие)
=======================================================

Положите сюда DLL из SDK Hantek, РАЗДЕЛЁННЫЕ ПО РАЗРЯДНОСТИ:

  hantek_dll/x64/   — если запускаете 64-битный Python
  hantek_dll/x86/   — если запускаете 32-битный Python

Минимально нужен HTHardDll.dll (связь с прибором). Рекомендуется положить рядом
все DLL из SDK: HTHardDll.dll, HTSoftDll.dll, HTDisplayDll.dll, HTMeasDll.dll и их
зависимости — кладите ВСЕ файлы из соответствующей разрядности.

ВАЖНО про разрядность:
* Разрядность DLL обязана совпадать с разрядностью Python. SDK Hantek часто
  поставляется 32-битным. Если у вас 64-битный Python, а DLL только 32-битная —
  либо возьмите 64-битную DLL, либо запустите приложение 32-битным Python.
* При несовпадении в журнале появится понятное сообщение (ERROR_BAD_EXE_FORMAT).

Куда ещё можно положить DLL (ищется в этом порядке):
  1) переменная окружения HANTEK_DLL_DIR (или HANTEK_DLL_DIR/<arch>);
  2) папка, указанная в меню «Приборы → Указать папку с DLL Hantek…»
     (сохраняется в ~/.signals/settings.json → "hantek_dll_dir");
  3) эти папки hantek_dll/x64 | hantek_dll/x86;
  4) <папка приложения>/drivers/x64 | x86;
  5) рядом с приложением или в текущем каталоге.

После размещения DLL нажмите «Обновить список приборов» — в журнал будет
выведено, какая DLL загружена (и её разрядность) и какие приборы откликнулись.
